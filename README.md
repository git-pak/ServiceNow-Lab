# ServiceNow ITSM Lab – Incident Management & Automation

## Overview
This repository showcases hands-on experience with ServiceNow ITSM workflows, focusing on incident management, structured troubleshooting, and workflow optimization.

The lab simulates a real-world help desk scenario where a user is unable to access corporate email on an iPhone. The issue is investigated using a systematic troubleshooting approach and resolved by resetting the device’s network settings.

---

## Objectives
- Simulate end-to-end incident lifecycle in ServiceNow
- Perform structured troubleshooting as a help desk technician
- Document technical findings and resolution steps
- Create Knowledge Base articles for repeat issues
- Explore workflow and automation improvements

---

## Environment
- **Platform:** ServiceNow Personal Developer Instance (PDI)
